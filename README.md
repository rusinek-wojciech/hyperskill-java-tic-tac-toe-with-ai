# Tic-Tac-Toe with AI

Famous tic-tac-toe game implemented in java with different difficulties and settings. 

Commands:
- start (x mode) (o mode) 
- exit 

Available modes:
- easy - full random moves
- medium - half random half ai
- hard - full ai, minimax algorithm+
- user - moves controlled by player
